# <a name="measuring-the-business-value-of-ajax"></a>Medir el valor empresarial de AJAX

por Eric Lee

ASP.NET AJAX es muy conocido por su capacidad para mejorar la experiencia del usuario de las aplicaciones Web. Con Visual Studio Team System también puede medir el valor empresarial de agregar ASP.NET AJAX a las aplicaciones Web. En concreto, puede usar las capacidades de carga y prueba Web de Visual Studio Team System para explorar cómo la diferencia en el tamaño de transmisión de datos HTTP afecta a las características de rendimiento y escalabilidad de la aplicación Web.
