## <a name="avoiding-cross-site-scripting-xss-attacks"></a><span data-ttu-id="cf943-152">Evitar ataques de Scripting entre sitios (XSS)</span><span class="sxs-lookup"><span data-stu-id="cf943-152">Avoiding Cross-Site Scripting (XSS) Attacks</span></span>

<span data-ttu-id="cf943-153">Cada vez que se aceptan la entrada de un usuario y volver a mostrar esa entrada en el sitio Web, podría abrir su sitio Web a los ataques de Scripting entre sitios (XSS).</span><span class="sxs-lookup"><span data-stu-id="cf943-153">Whenever you accept input from a user, and redisplay that input on your website, you potentially open your website to Cross-Site Scripting (XSS) Attacks.</span></span> <span data-ttu-id="cf943-154">En teoría, un usuario malintencionado podría enviar código de JavaScript que se ejecuta cuando se vuelve a mostrar la entrada.</span><span class="sxs-lookup"><span data-stu-id="cf943-154">In theory, a malicious hacker could submit JavaScript code that gets executed when the input is redisplayed.</span></span> <span data-ttu-id="cf943-155">El código JavaScript podría usarse para robar contraseñas de usuario u otra información confidencial.</span><span class="sxs-lookup"><span data-stu-id="cf943-155">The JavaScript could be used to steal user passwords or other sensitive information.</span></span>

<span data-ttu-id="cf943-156">Normalmente, puede anular los ataques XSS de codificación cualquier entrada que recuperar de un usuario antes de mostrarlos en una página web HTML.</span><span class="sxs-lookup"><span data-stu-id="cf943-156">Normally, you can defeat XSS attacks by HTML encoding whatever input you retrieve from a user before displaying it in a web page.</span></span> <span data-ttu-id="cf943-157">Sin embargo, la salida del Editor HTML de codificación HTML no sólo codificaría &lt;script&gt; etiquetas, también codificaría todas las etiquetas HTML.</span><span class="sxs-lookup"><span data-stu-id="cf943-157">However, HTML encoding the output of the HTML Editor would not only encode &lt;script&gt; tags, it would also encode all HTML tags.</span></span> <span data-ttu-id="cf943-158">En otras palabras, se pierden todos los formatos, como el tipo de fuente, tamaño de fuente y color de fondo.</span><span class="sxs-lookup"><span data-stu-id="cf943-158">In other words, you would lose all of the formatting such as the font type, font size, and background color.</span></span>
