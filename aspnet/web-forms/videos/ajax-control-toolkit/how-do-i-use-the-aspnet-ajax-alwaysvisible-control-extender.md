# <a name="how-do-i-use-the-aspnet-ajax-alwaysvisible-control-extender"></a>[Cómo:] ¿Usar el extensor de control AlwaysVisible de ASP.NET AJAX?

por [Joe Stagner](https://github.com/JoeStagner)

Vea lo fácil que es delimitar un elemento de página a una posición en la pantalla, donde permanece siempre visible sin importar hasta qué punto el usuario puede desplazarse por el documento. El elemento de página flotante puede ser cualquier cosa desde texto estático hasta un cuadro de ayuda basado en AJAX y contextual.
