# <a name="getting-started-with-reporting-services"></a>Introducción a Reporting Services

por [Microsoft](https://github.com/microsoft)

SQL Server Reporting Services permite diseñar e implementar informes interactivos y con un formato bonito de los datos. Adecuado para la distribución en línea o de impresión, estos informes y le permiten obtener resultados profesionales con mucha rapidez. En este vídeo se muestra cómo configurar Reporting Services y el Business Intelligence Development Studio para generar informes con los asistentes y diseñadores, e implementarlos en el servidor de informes.
