# <a name="code-coverage-of-automated-tests"></a>Cobertura de código de pruebas automatizadas

por Eric Lee

En este vídeo se muestra cómo usar la cobertura de código como una métrica para determinar la eficacia de las pruebas en una aplicación Web ASP.NET. Una vez completada una prueba, Visual Studio Team System informa del porcentaje de código que se ha ejecutado e indica código ejecutado con un fondo verde y código no ejecutado con un fondo rojo, lo que facilita la visualización del código que se encontraba en la prueba. También obtendrá información sobre cómo puede usar C# o Visual Basic para extender la prueba automatizada.
