## <a name="avoiding-cross-site-scripting-xss-attacks"></a><span data-ttu-id="eb109-152">Evitar ataques de secuencias de comandos entre sitios (XSS)</span><span class="sxs-lookup"><span data-stu-id="eb109-152">Avoiding Cross-Site Scripting (XSS) Attacks</span></span>

<span data-ttu-id="eb109-153">Siempre que acepte la entrada de un usuario y vuelva a mostrares esa entrada en su sitio web, es posible que abra su sitio web en ataques de secuencias de comandos entre sitios (XSS).</span><span class="sxs-lookup"><span data-stu-id="eb109-153">Whenever you accept input from a user, and redisplay that input on your website, you potentially open your website to Cross-Site Scripting (XSS) Attacks.</span></span> <span data-ttu-id="eb109-154">En teoría, un hacker malintencionado podría enviar código JavaScript que se ejecuta cuando se vuelve a mostrar la entrada.</span><span class="sxs-lookup"><span data-stu-id="eb109-154">In theory, a malicious hacker could submit JavaScript code that gets executed when the input is redisplayed.</span></span> <span data-ttu-id="eb109-155">El JavaScript podría ser utilizado para robar contraseñas de usuario u otra información confidencial.</span><span class="sxs-lookup"><span data-stu-id="eb109-155">The JavaScript could be used to steal user passwords or other sensitive information.</span></span>

<span data-ttu-id="eb109-156">Normalmente, puede derrotar los ataques XSS codificando cualquier entrada que recupere de un usuario antes de mostrarla en una página web.</span><span class="sxs-lookup"><span data-stu-id="eb109-156">Normally, you can defeat XSS attacks by HTML encoding whatever input you retrieve from a user before displaying it in a web page.</span></span> <span data-ttu-id="eb109-157">Sin embargo, la codificación HTML de &lt;la&gt; salida del Editor HTML no solo codificaría las etiquetas de script, sino que también codificaría todas las etiquetas HTML.</span><span class="sxs-lookup"><span data-stu-id="eb109-157">However, HTML encoding the output of the HTML Editor would not only encode &lt;script&gt; tags, it would also encode all HTML tags.</span></span> <span data-ttu-id="eb109-158">En otras palabras, perdería todo el formato, como el tipo de fuente, el tamaño de fuente y el color de fondo.</span><span class="sxs-lookup"><span data-stu-id="eb109-158">In other words, you would lose all of the formatting such as the font type, font size, and background color.</span></span>
