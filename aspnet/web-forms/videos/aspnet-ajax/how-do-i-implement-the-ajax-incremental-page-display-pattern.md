# <a name="how-do-i-implement-the-ajax-incremental-page-display-pattern"></a>[Cómo:] ¿Implementar el patrón de presentación de página incremental de AJAX?

por [Joe Stagner](https://github.com/JoeStagner)

Tradicionalmente, cuando se solicita una página web, el usuario debe esperar mientras se recupera toda la página del servidor después de que la página completa aparezca repentinamente en el explorador. En este vídeo se muestra cómo el patrón de presentación de página incremental de AJAX mejora la experiencia del usuario al capturar y mostrar de forma rápida y sencilla partes de la Página Web.
