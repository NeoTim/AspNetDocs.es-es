# <a name="how-do-i-use-the-aspnet-ajax-maskededit-controls"></a>[Cómo:] ¿Usar los controles MaskedEdit de AJAX de ASP.NET?

por [Joe Stagner](https://github.com/JoeStagner)

El kit de herramientas de control de AJAX de ASP.NET proporciona el control MaskedEdit y el control MaskedEditValidator, que funcionan conjuntamente para solicitar, aplicar estilo y validar los datos que el usuario escribe en un cuadro de texto. Esto permite que la Página Web proporcione al usuario información visual inmediata sobre si los datos especificados tienen un tipo y un formato válidos.
