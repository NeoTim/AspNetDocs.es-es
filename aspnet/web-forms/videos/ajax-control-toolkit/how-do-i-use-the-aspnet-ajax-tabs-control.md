# <a name="how-do-i-use-the-aspnet-ajax-tabs-control"></a>[Cómo:] ¿Usar el control pestañas de ASP.NET AJAX?

por [Joe Stagner](https://github.com/JoeStagner)

En este vídeo se muestran los aspectos básicos del control de pestañas del kit de herramientas de control de AJAX de ASP.NET, con el que puede crear un conjunto de pestañas para organizar el contenido en una página web. La pestaña seleccionada se mantiene entre copias de seguridad, lo que hace que el control de pestañas sea idóneo para las páginas de administración basadas en formularios.
