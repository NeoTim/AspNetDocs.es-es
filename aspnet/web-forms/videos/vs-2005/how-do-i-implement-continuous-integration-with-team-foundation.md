# <a name="how-do-i-implement-continuous-integration-with-team-foundation"></a>[¿Cómo lo hago?:] ¿Implementar la integración continua con Team Foundation?

por [Chris Menegay](https://twitter.com/CMenegay)

Hay una serie de herramientas de integración continua que funcionen sobre Team Foundation Server y, en este vídeo veremos la versión gratuita [Norton TeamCI](http://teamsystemrocks.com/files/12/tools/entry1018.aspx) herramienta disponible en [Team System Rocks](http://teamsystemrocks.com/). TeamCI comprueba periódicamente si actualiza los archivos se han protegido en Team Foundation Server y, si es así, se compile la solución y se ejecutarán las pruebas automatizadas.
