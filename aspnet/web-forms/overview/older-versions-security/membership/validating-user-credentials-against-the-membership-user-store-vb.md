### <a name="how-the-membership-framework-handles-invalid-login-attempts"></a><span data-ttu-id="f2a70-150">Cómo controla el marco de pertenencia los intentos de inicio de sesión no válidos</span><span class="sxs-lookup"><span data-stu-id="f2a70-150">How the Membership Framework Handles Invalid Login Attempts</span></span>

credentials are valid, the HTTP response includes the authentication ticket in a cookie.</span></span> <span data-ttu-id="f2a70-153">Por lo tanto, un pirata informático que intente interrumpir su
