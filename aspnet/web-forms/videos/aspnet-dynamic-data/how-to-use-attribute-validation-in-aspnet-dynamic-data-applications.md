# <a name="how-to-use-attribute-validation-in-aspnet-dynamic-data-applications"></a>Cómo usar la validación de atributos en aplicaciones de datos dinámicos de ASP.NET

por [Joe Stagner](https://github.com/JoeStagner)

Este vídeo introducirá la modificación de las clases parciales en la capa de acceso a datos de la aplicación datos dinámicos y empezaremos a usar los atributos centrándose en los atributos de validación. Esta validación de capa de acceso a datos es preferible a la validación en el nivel de interfaz de usuario y es muy fácil de hacer con ASP.NET datos dinámicos.
