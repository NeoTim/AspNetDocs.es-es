# <a name="how-do-i-implement-view--models-to-manage-data-for-aspnet-mvc-views"></a>Cómo: implementar modelos de vista para administrar datos para las vistas de ASP.NET MVC

por [Chris Pels](https://twitter.com/chrispels)

En este vídeo, Chris Pels muestra cómo usar el concepto de "modelo de vista" para administrar datos para las vistas de ASP.NET MVC. En primer lugar, se crea un proyecto de MVC de ejemplo que usa ViewDataDictionary para empaquetar y pasar datos de un controlador a una vista. A continuación, se crea una clase que representa un modelo de vista para una vista en el proyecto de MVC de ejemplo. Como parte de esta implementación, se tratan las ventajas del enfoque de modelo de vista. Por último, se explica cómo particionar la lógica de la aplicación cuando se usa una implementación de modelo de vista.
