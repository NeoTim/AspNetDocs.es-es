# <a name="webmatrix-content-guide"></a>Guía de contenido de WebMatrix

> [!NOTE] 
> WebMatrix ya no se recomienda como un entorno de desarrollo integrado para ASP.NET Web Pages. Use [Visual Studio](xref:aspnet/web-pages/overview/getting-started/program-asp-net-web-pages-in-visual-studio) o [Visual Studio Code](https://code.visualstudio.com/).

WebMatrix es un conjunto de herramientas que proporciona una manera sencilla de compilar sitios web para Microsoft Windows. Incluye IIS Express (un servidor Web de desarrollo), ASP.NET (un marco de programación web) y SQL Server Compact (una base de datos incrustada). Para ver una visita rápida de WebMatrix, consulte [información general sobre WebMatrix](https://www.microsoft.com/web/webmatrix/). Para revisar las características principales de WebMatrix, consulte [características de WebMatrix](https://www.microsoft.com/web/webmatrix/features/).
