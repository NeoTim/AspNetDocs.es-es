# <a name="how-do-i-tune-web-application-performance-with-profiling"></a>[Cómo:] Optimizar el rendimiento de las aplicaciones web con la generación de perfiles

por [Chris Menegay](https://twitter.com/CMenegay)

En este vídeo aprendemos que Visual Studio Team System proporciona dos métodos para generar perfiles de una aplicación web: muestreo e instrumentación. La instrumentación inyecta marcadores en el código subyacente para que el desarrollador pueda ver todos los detalles de la ejecución de la aplicación, incluidos los árboles de llamadas y la asignación de objetos, para ayudar a optimizar el rendimiento de la aplicación.
