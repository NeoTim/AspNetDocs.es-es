# <a name="multi-targeting-support-in-visual-studio-2008"></a>Compatibilidad con múltiples versiones en Visual Studio 2008

por [Joe Stagner](https://github.com/JoeStagner)

Mostraré la característica de compatibilidad con múltiples versiones de Visual Studio 2008. En el pasado, tenía que elegir la versión de Visual Studio para que coincida con la versión de .NET de destino. Por ejemplo, Visual Studio 2005 funcionó con .NET 2,5. Ahora, con Visual Studio 2008 puede elegir la versión de .NET adecuada para su aplicación.
