# <a name="the-datapager-control"></a>Control de paginación de controles

por [Joe Stagner](https://github.com/JoeStagner)

Por lo general, el control de paginación se usa con el control ListView. El control ListView está controlado por datos. En primer lugar, cree un control ListView y proporcione algunos datos. Después, coloque el control de paginación de página en la página para mejorar la funcionalidad de la Página Web. Si no ha observado el vídeo de procedimientos de control de ListView, sería mejor revisarlo antes de usar el control de administrador de controles de página.
