# <a name="measuring-the-business-value-of-ajax"></a>Medir el valor empresarial de AJAX

por Eric Lee

ASP.NET AJAX es conocido por su capacidad para mejorar la experiencia del usuario de aplicaciones Web. Con Visual Studio Team System puede medir también el valor empresarial de adición de ASP.NET AJAX a sus aplicaciones Web. En concreto puede usar las capacidades de carga y las pruebas Web de Visual Studio Team System para explorar cómo afecta la diferencia de tamaño de transmisión de datos HTTP a las características de rendimiento y escalabilidad de la aplicación Web.
