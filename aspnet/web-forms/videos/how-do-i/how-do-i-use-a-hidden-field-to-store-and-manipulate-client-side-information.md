<a name="how-do-i-use-a-hidden-field-to-store-and-manipulate-client-side-information"></a>[¿Cómo lo hago?:] Usar un campo oculto para Store y manipular información del lado cliente
====================
por [Chris Pels](https://twitter.com/chrispels)

En este vídeo Chris Pels mostrará cómo usar un campo oculto como un enfoque a la administración de estado simple en una página ASP.NET y para comunicación de cliente información del lado del procesamiento. En primer lugar, consulte cómo agregar un campo oculto a una página mediante un control de servidor ASP.NET HiddenField y recuperar el valor del campo oculto en el código de servidor después de una devolución de datos. A continuación, obtenga información sobre cómo manipular el valor del campo oculto en el script de cliente y, a continuación, recupere el valor modificado en el código del lado servidor. Por último, ver un ejemplo de cómo usar varios campos ocultos para almacenar información de procesamiento del lado cliente que se puede registrar para el procesamiento server.
