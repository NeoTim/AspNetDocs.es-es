# <a name="aspnet-docs"></a>ASP.NET Docs

Este repositorio contiene la documentación conceptual ASP.NET 4. x hospedada en [docs.Microsoft.com/ASPNET](https://docs.microsoft.com/aspnet). Vea la [Guía de colaboración](CONTRIBUTING.md) y la [lista de problemas](https://github.com/dotnet/AspNetDocs/issues) si quiere ayudar.

Los cambios en la documentación de la API deben realizarse en el [repositorio ApiDocs](https://github.com/aspnet/ApiDocs) frente a los comentarios `///` con triple barra diagonal.
