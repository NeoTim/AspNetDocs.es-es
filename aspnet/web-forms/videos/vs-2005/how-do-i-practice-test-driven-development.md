# <a name="how-do-i-practice-test-driven-development"></a>[Cómo:] ¿Practicar el desarrollo controlado por pruebas?

por [Chris Menegay](https://twitter.com/CMenegay)

Visual Studio Team System proporciona las herramientas necesarias para practicar el desarrollo controlado por pruebas, y este vídeo aprenderá cómo la funcionalidad de pruebas de Team System le permite ir desde el código para probar y probar el código. Vemos cómo puede escribir una prueba y, a continuación, usar Visual Studio Team System para generar el código de destino, que se adhiere al Ideology de TDD de escribir primero la prueba y el código segundo.
