# <a name="how-do-i-synchronize-data-in-secondary-user-controls-based-upon-user-selections"></a>[Cómo]: sincronizar datos en controles de usuario secundarios en función de las selecciones del usuario

por [Chris Pels](https://twitter.com/chrispels)

En este vídeo, Chris Pels muestra cómo crear dos controles DropDownList donde al seleccionar un elemento de la primera lista se rellenan los elementos de la segunda lista. En primer lugar, se crea una página ASP.NET con dos controles DropDownList, uno para Customer y el otro para los Estados en los que el cliente hace negocios. A continuación, la lista de clientes se rellena en el código subyacente. Después, en el evento SelectedIndexChanged de la lista de clientes, los elementos de la lista de Estados se rellenan en función de la selección. Por último, hay una explicación de cómo se puede aplicar este enfoque a otros tipos de controles y datos.
