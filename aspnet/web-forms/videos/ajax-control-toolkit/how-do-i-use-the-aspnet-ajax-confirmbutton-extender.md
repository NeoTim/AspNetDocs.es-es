# <a name="how-do-i-use-the-aspnet-ajax-confirmbutton-extender"></a>[Cómo:] ¿Usar el extensor ConfirmButton de ASP.NET AJAX?

por [Joe Stagner](https://github.com/JoeStagner)

Obtenga información sobre cómo el extensor ConfirmButton en el kit de herramientas de control de AJAX de ASP.NET proporciona una manera sencilla de agregar un cuadro de diálogo de confirmación al evento click de un control Button, LinkButton o ImageButton. Esto proporciona al usuario la opción de cancelar una acción inminente.
