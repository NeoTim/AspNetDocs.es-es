# <a name="how-do-i-implement-the-ajax-after-processing-pattern"></a>[Cómo:] ¿Implementar AJAX después del patrón de procesamiento?

por [Joe Stagner](https://github.com/JoeStagner)

El marco de ASP.NET AJAX permite que el código del lado servidor se enlace y responda a varios eventos del lado cliente. En este vídeo aprenderá a implementar el patrón después del procesamiento, que permite que el código del lado servidor ejecute la lógica de ajuste una vez que el usuario haya terminado con la Página Web.
