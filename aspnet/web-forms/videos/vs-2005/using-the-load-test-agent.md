<a name="using-the-load-test-agent"></a>Mediante el agente de prueba de carga
====================
por Eric Lee

En este vídeo se examina cómo incorporar al agente de prueba de carga en la prueba que realizamos con Visual Studio Team System. Un agente de prueba de carga es parte de una "plataforma" que se puede implementar para separar las máquinas para habilitar la carga, esfuerzo y hacer pruebas unitarias partir otra configuración de máquina o del sistema operativo. Cada plataforma puede trabajar con varios agentes y se puede implementar en máquinas virtuales o remotas, lo que permite las pruebas completas con Visual Studio Team System.
