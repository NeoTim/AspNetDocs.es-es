# <a name="how-do-i-use-the-aspnet-ajax-dropdown-control"></a>[Cómo:] ¿Usar el control desplegable ASP.NET AJAX?

por [Joe Stagner](https://github.com/JoeStagner)

Con el extensor del control DropDown de ASP.NET AJAX control Toolkit, puede presentar al usuario una lista desplegable de opciones que desencadenan una actualización de página parcial en otra parte de la Página Web. La opción seleccionada por el usuario se pasa al servidor, lo que permite que el código del lado servidor procese la selección antes de completar la actualización parcial de la página.
